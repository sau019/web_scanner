# Basic Python Port Scanner

A simple TCP port scanner written in Python for learning networking and cyber security fundamentals.

## Features
- Scan TCP ports (OPEN / CLOSED)
- Supports IP address and domain name
- Shows common service names (FTP, SSH, HTTP, etc.)
- Scan summary at the end

## Requirements
- Python 3.x
- Works on Windows and Linux (Kali)

## Usage
```bash
python3 port_scanner.py
